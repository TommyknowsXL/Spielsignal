# Täglicher redaktioneller Ablauf

1. GitHub öffnen.
2. Actions öffnen.
3. Workflow `Daily Editorial Queue` auswählen.
4. Letzten Lauf öffnen.
5. Tagesbericht als Artefakt herunterladen oder Bericht im Repository prüfen.
6. Top-10-Vorschläge ansehen.
7. Interessante Kandidaten auswählen.
8. Bildquelle prüfen.
9. Bildstatus manuell auf `approved` setzen, falls Nutzungsgrundlage dokumentiert ist.
10. Codex beauftragen, aus ausgewählten Vorschlägen Artikelentwürfe zu erstellen.
11. Artikel selbst prüfen.
12. Erst danach veröffentlichen.

Ein grüner Workflow-Lauf bedeutet nur, dass Quellenabruf, Bericht, Tests und Build technisch
funktioniert haben. Er ist keine redaktionelle oder rechtliche Freigabe.
