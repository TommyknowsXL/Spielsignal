# Täglicher redaktioneller Ablauf

1. GitHub öffnen.
2. Actions öffnen.
3. Workflow `Daily Editorial Queue` auswählen.
4. Letzten Lauf öffnen.
5. Tagesbericht als Artefakt herunterladen oder Bericht im Repository prüfen.
6. Top-10-Vorschläge und Steam-Scout-Status ansehen.
7. Interessante Kandidaten auswählen.
8. Spielname, Gratis-Klassifikation und gegebenenfalls Steam-App-ID prüfen.
9. Bildquelle und Bildkandidat prüfen.
10. Bildstatus manuell auf `approved` setzen, falls die Nutzungsgrundlage dokumentiert ist.
11. Codex beauftragen, aus ausgewählten Vorschlägen Artikelentwürfe zu erstellen.
12. Artikel selbst prüfen.
13. Erst danach veröffentlichen.

Ein grüner Workflow-Lauf bedeutet nur, dass Quellenabruf, Bericht, Tests und Build technisch
funktioniert haben. Er ist keine redaktionelle oder rechtliche Freigabe.
